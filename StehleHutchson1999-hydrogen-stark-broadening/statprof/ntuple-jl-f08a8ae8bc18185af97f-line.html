55 Create a tuple of length `N`, computing each element as `f(i)`,
56 where `i` is the index of the element. By taking a `Val(N)`
57 argument, it is possible that this version of ntuple may
58 generate more efficient code than the version taking the

60 `ntuple(f, Val(N))` in cases where `N` cannot be determined
61 at compile time.

63 # Examples
